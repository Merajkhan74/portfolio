# 🚀 Meraj Khan - Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and frontend development expertise. Designed with a strong focus on clean UI, smooth user experience, and real-world inspired design systems.

---

## 🌐 Live Demo

🔗 https://portfolio-one-cyan-15.vercel.app/

---

## ✨ Features

* ⚡ Fully responsive design (mobile, tablet, desktop)
* 🎯 Clean and modern user interface
* 🔄 Smooth scrolling with hidden scrollbar
* 🧩 Reusable component-based architecture
* 📁 Well-structured project showcase
* 🚀 Performance optimized for fast loading

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Language:** JavaScript (ES6+)

---

## ⚙️ Installation

```bash
git clone https://github.com/Merajkhan74/portfolio.git
cd portfolio
npm install
npm start
```
